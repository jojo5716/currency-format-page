CurrencyFormat.js Is a tiny JavaScript library, providing simple and advanced money and currency formatting

Usage
  1. Download the script and put it in your project, then reference it in your HTML.
    <script type="text/javascript" src="https://raw.githubusercontent.com/jojo5716/CurrencyFormat/master/currency-format.js"></script>
  2. Download the script and put it in your project, then reference it in your HTML.
    var price = currencies.formatMoney();
    // € 0.00
    var price = currencies.formatMoney(2000.65, 'EUR');
    // € 2,000.65
    var price = currencies.formatMoney(2000.65, 'EUR', 'es_ES');
    // 2.000,65 €
    var price = currencies.formatMoney(2000.65, 'EUR', 'es_ES', 3);
    // 2.000,650 €
    var price = currencies.formatMoney(2000.65, 'EUR', 'es_ES', 3, ',');
    // 2,000,650 €
    var price = currencies.formatMoney(2000.65, 'EUR', 'es_ES', 3, ',', '.');
    // 2,000.650 €
